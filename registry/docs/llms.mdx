The dynamically-generated [`/llms.txt`](/llms.txt) route can be used by LLMs for a compact map of the docs, registry sections, and item pages.

Similarly, a [`/llms-full.txt`](/llms-full.txt) is available for AI tools that require expanded context file with the generated Markdown content inlined.

Every docs page, registry section, and registry item also has an LLM-friendly `.md` route. Try [`/docs.md`](/docs.md), [`/components.md`](/components.md), or [`/components/example-card.md`](/components/example-card.md).

---

Install a registry item with the command shown on its docs page.

```bash
npx shadcn@latest add https://underscore-cn.vercel.app/example-card.json
```

The command uses the registry item URL generated from `siteConfig.homepage` and `siteConfig.registry`.

Use the package manager selector on item pages to switch between npm, pnpm, yarn, and bun commands.

This template includes an installable Agent Skill that teaches coding agents how to author `_cn` registry items.

Install it from the upstream template:

```
npx skills add jakejarvis/_cn --skill shadcn-registry
```

Forks can install from their own repository instead:

```
npx skills add <owner>/<repo> --skill shadcn-registry
```

After installing the skill, ask for registry work directly:

- "add a button component to the registry"
- "adapt this modal from my app to make it reusable via this shadcn registry"
- "add a reusable hook to the registry"
- "turn this dashboard section into a registry block"

The skill covers components, blocks, hooks, libs/helpers, pages, files, previews, usage docs, and dependency metadata.
